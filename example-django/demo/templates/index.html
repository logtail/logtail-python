About this Demo

This demo shows how to integrate Logtail with Django for structured logging. Each button above triggers a different type of log:

  • Info Log: Logs basic information about the request including the user agent
  • Warning Log: Demonstrates logging with custom data structures
  • Error Log: Shows how exceptions are logged with full stack traces

Check your Logtail dashboard to see the logged events!
